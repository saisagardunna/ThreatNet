# ThreatNet - Cyber Threat Intelligence Platform

ThreatNet is an advanced AI-powered Cyber Threat Intelligence (CTI) dashboard designed to detect and analyze cybersecurity threats in real-time.

## Features
- **AI-Powered Analysis:** Uses Groq API (Llama-3 model) for advanced threat detection.
- **Dataset Integration:** References a local cyber threat dataset for known signatures.
- **Real-time Dashboard:** React-based frontend for monitoring and analysis.
- **PDF Reports:** Detailed downloadable reports for analyzed threats.
- **Cute Bot Assistant:** Interactive AI chatbot for user guidance.

## Tech Stack
- **Frontend:** React, Vite, Framer Motion, Lottie React
- **Backend:** Python, FastAPI, Pandas, Scikit-learn
- **AI Engine:** Groq API, Local ML Models

## Setup
1. Clone the repository.
2. Install frontend dependencies: `cd web_app/frontend && npm install`
3. Install backend dependencies: `cd web_app/backend && pip install -r requirements.txt`
4. Run the app!
